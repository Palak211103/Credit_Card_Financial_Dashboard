
#  Real-Time Credit Card Financial Dashboard

##  Project Overview
This project is a real-time financial analytics dashboard designed to analyze credit card transactions and customer data. It provides actionable insights through interactive KPIs, gauges, and visualizations for effective financial monitoring.

##  Objectives
- Track real-time revenue and transaction metrics
- Analyze customer behavior and spending patterns
- Visualize insights by state, education level, and card type
- Support data-driven financial decisions

##  Key Features
- Real-time KPI cards (Revenue, Amount, Transactions)
- Gauge charts for performance monitoring
- Card type-based financial analysis
- Education level-wise customer insights
- State-wise revenue and transaction analysis
- Interactive and dynamic dashboard views

##  Datasets Used
- Credit Card Customer Report
- Credit Card Transaction Report

##  Tools & Technologies
- Data Visualization: Tableau / Power BI
- Data Processing: SQL, Excel
- Dashboard Type: Real-Time Financial Dashboard

##  Insights Generated
- Identifies high-revenue states and regions
- Highlights spending trends by card type
- Shows revenue distribution across education levels
- Monitors real-time financial performance
